Sistema básico para el registro y administración de la documentación de clientes de **Dolorfin**.

## Descripción
Aplicación de escritorio que permite gestionar los datos y documentos asociados a cada cliente de forma organizada.
# Dolorfin — Sistema de Gestión de Clientes para Consultorios

Aplicación de escritorio para la gestión de clientes/pacientes en consultorios y clínicas. Permite registrar información de pacientes, adjuntar documentos (imágenes, PDFs), realizar búsquedas avanzadas y generar respaldos de la base de datos.

## Público objetivo

Profesionales de la salud (fisioterapeutas, quiroprácticos, especialistas en manejo del dolor) que requieren un sistema local, simple y en español para administrar expedientes de pacientes.

## Funcionalidades

- **CRUD de clientes:** Registrar, editar, consultar y eliminar pacientes con nombre, apellido, género, fecha de inserción, fecha de documento y tipo de trabajo (Valoración / Servicio).
- **Gestión de documentos:** Adjuntar múltiples archivos (imágenes y PDFs) por paciente. Previsualización de imágenes con miniaturas y apertura de PDFs con la aplicación predeterminada del sistema.
- **Búsqueda avanzada:** Filtrar clientes por nombre, apellido, fechas y tipo de trabajo desde la ventana de consulta.
- **Backup y restauración:** Exportar toda la base de datos y archivos adjuntos a un archivo ZIP, e importar respaldos existentes.
- **Interfaz visual personalizada:** Tema neón morado con efectos de sombra, completamente en español.
